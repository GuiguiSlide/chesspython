# Projet Échecs Python

Ce projet est un jeu d'échecs développé en Python.

## Prérequis

- Python 3.x
- Librairie nécessaire : `ursina`

## Installation

1. Clonez ce dépôt.
2. Installez la librairie requise :
    ```bash
    pip install ursina
    ```

## Utilisation

Lancez le fichier principal pour démarrer le jeu :
```bash
python main.py
```
